# 🏦 Loan Management System

A console-based Loan Management System developed in C++ using Object-Oriented Programming and File Handling concepts.

## 📌 Overview

This project helps banks or financial institutions manage loan applications, calculate EMI, track loan status, and maintain customer records.

The system supports multiple loan categories and stores information permanently using text files.
